---

### Download

    git clone https://github.com/reidiiius/kasterborous.git

---

### Usage
command line

    cd kasterborous

    octave -qf gallifrey.m | less

or filter with grep

    octave -qf gallifrey.m | grep -B1 -A8 'n0'

set execute permission

    chmod u+x gallifrey.m

    ./gallifrey.m | less
